## 四、交互细节说明

### 1. 单项编辑限制

同一时间只能编辑一个关键词或公众号：

```

┌─────────────────────────────────────────────────┐
│  [  AI  ]                      [✓] [✗]         │

❌ 错误（不会出现）：
┌─────────────────────────────────────────────────┐
│  [  AI  ]                      [✓] [✗]         │
│  [  科技  ]                    [✓] [✗]         │
```

### 2. 视觉反馈

**编辑模式**：
- 输入框有强调色边框（绿色）
- 保存按钮为绿色勾选图标

**显示模式**：
- 文本正常显示

## 五、使用技巧

### 技巧1：快速修正错误

如果发现输入错误，立即点击编辑修改，无需删除重新添加。

### 技巧2：批量调整

虽然不支持批量编辑，但可以快速逐个修改：
1. 点击第一个项的编辑图标
2. 修改后点击保存
3. 立即点击下一个项的编辑图标
4. 重复操作

### 技巧3：预览后保存

在输入框中修改后，可以先预览效果，确认无误后再点击保存。

### 技巧4：使用取消功能

如果修改后不满意，点击取消图标恢复原始文本，无需重新输入。

## 六、常见使用场景

### 场景1：修正输入错误

**问题**：添加关键词时输入错误，例如"人工只能"应为"人工智能"

**解决**：
1. 点击"人工只能"右侧的编辑图标
2. 修改为"人工智能"
3. 点击保存

### 场景2：公众号改名

**问题**：关注的公众号改名了，例如"AI前线"改为"AI科技前线"

**解决**：
1. 点击"AI前线"右侧的编辑图标
2. 修改为"AI科技前线"
3. 点击保存
4. 刷新文章列表，使用新名称搜索

### 场景3：优化搜索效果

**问题**：关键词"AI"搜索结果太泛，想改为更具体的"AI大模型"

**解决**：
1. 点击"AI"右侧的编辑图标
2. 修改为"AI大模型"
3. 点击保存
4. 刷新文章列表，获取更精准的结果

## 七、注意事项

### ⚠️ 重要提醒

1. **历史数据不变**：编辑关键词或公众号后，历史文章仍保留原名称
2. **需要重新抓取**：修改后建议刷新文章列表，使用新名称搜索
3. **一次一个**：同一时间只能编辑一个项，确保操作清晰
4. **不能为空**：关键词和公众号名称不能为空
5. **自动去空格**：保存时会自动去除首尾空格

### 💡 最佳实践

1. **谨慎修改**：确认需要修改后再操作，避免频繁改动
2. **及时刷新**：修改后及时刷新文章列表，查看新结果
3. **记录变更**：如果修改较多，建议记录变更内容
4. **测试效果**：修改后测试搜索效果，确认是否达到预期
