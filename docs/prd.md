# 需求文档

## 1. 应用概述

- **应用名称**：公众号内容聚合助手
- **应用描述**：一款轻量化的微信小程序，支持用户自定义关键词与公众号监控源，定时聚合相关最新文章简报，并通过微信服务通知在指定时间推送给用户。

---

## 2. 用户与使用场景

### 2.1 目标用户
- 需要定期追踪特定公众号内容动态的个人用户
- 关注 AI、科技等领域资讯的信息消费者

### 2.2 核心使用场景
- 用户配置关键词与公众号监控列表，每天下午 6 点自动接收聚合简报推送
- 用户在小程序内查看各公众号最新文章摘要，快速了解信息动态

---

## 3. 页面结构与功能说明

### 3.1 整体页面结构

```
公众号内容聚合助手
├── 首页（简报列表页）
├── 配置页
│   ├── 关键词管理
│   └── 公众号管理
└── 推送设置页
```

### 3.2 首页（简报列表页）

**功能说明：**
- 展示当日聚合的内容简报列表，按公众号或关键词分组展示
- 每个关键词下展示最多 5 条最新匹配文章，每条包含：
  - 文章标题
  - 所属公众号名称
  - 发布时间
  - 文章摘要（简短描述）
  - 跳转原文链接
- 支持手动刷新，重新拉取最新内容
- 若当日暂无新内容，展示「暂无新消息」提示

### 3.3 配置页

#### 3.3.1 关键词管理
- 展示当前已添加的关键词列表，初始预置关键词（用户可自行修改）
- 支持新增关键词（文本输入）
- 支持编辑已有关键词（点击关键词进入编辑态，可修改关键词文本，确认后保存）
- 支持删除已有关键词
- 关键词数量上限：10 个

#### 3.3.2 公众号管理
- 展示当前已添加的公众号列表，初始预置：量子位、赛博禅心、AI（用户可自行修改）
- 支持新增公众号（输入公众号名称）
- 支持编辑已有公众号名称（点击公众号进入编辑态，可修改公众号名称，确认后保存）
- 支持删除已有公众号
- 公众号数量上限：20 个

### 3.4 推送设置页

- 展示当前推送时间，默认为每天 18:00
- 支持修改推送时间（时间选择器，精确到小时）
- 推送方式固定为微信服务通知，不可更改
- 支持开启 / 关闭推送开关
- 用户首次进入时，引导授权微信服务通知权限

---

## 4. 业务规则与逻辑

1. **内容聚合规则**：系统每天在推送时间前完成内容抓取，按关键词匹配各公众号最新文章，每个关键词每个公众号最多取 5 条最新内容。
2. **推送触发规则**：每天到达用户设定推送时间时，若存在新内容（相较上次推送有更新），则通过微信服务通知推送简报摘要；若无新内容，则不推送。
3. **推送内容格式**：微信服务通知包含关键词名称、新增文章数量及首条文章标题，用户点击通知后跳转至小程序首页简报列表。
4. **登录方式**：微信授权登录 + 账号密码登录，用户配置数据与账号绑定，支持持久化存储。
5. **数据存储**：用户配置（关键词、公众号列表、推送时间）及历史简报数据均存储于后端数据库。
6. **编辑保存规则**：关键词或公众号编辑完成后，点击确认方可保存；编辑期间点击取消则放弃修改，恢复原始内容；编辑内容不得为空，为空时提示「内容不能为空」。

---

## 5. 异常与边界情况

| 场景 | 处理方式 |
|------|----------|
| 用户未授权微信服务通知 | 推送设置页提示引导授权，未授权则推送功能不可用 |
| 关键词或公众号列表为空 | 首页提示「请先前往配置页添加关键词和公众号」 |
| 内容抓取失败或超时 | 首页展示「内容获取失败，请稍后手动刷新」提示 |
| 关键词无匹配内容 | 对应关键词分组下展示「暂无相关内容」 |
| 关键词数量达上限 | 新增时提示「最多支持 10 个关键词」 |
| 公众号数量达上限 | 新增时提示「最多支持 20 个公众号」 |
| 编辑内容为空提交 | 提示「内容不能为空」，不允许保存 |

---

## 6. 验收标准

1. 用户可正常完成微信授权登录，配置数据持久化保存。
2. 关键词与公众号的增删改操作实时生效，页面正确刷新展示。
3. 每天 18:00 准时触发内容聚合与推送，有新内容时微信服务通知正常送达。
4. 首页简报列表按关键词分组展示，每组最多 5 条，文章信息完整（标题、来源、时间、摘要、链接）。
5. 无新内容时不触发推送，首页展示「暂无新消息」。
6. 推送时间修改后，次日起按新时间执行。
7. 关键词与公众号编辑功能正常，修改内容可正确保存，取消操作不影响原始数据。
8. 编辑内容为空时，保存操作被拦截并给出提示。
9. 异常场景下页面有明确提示，不出现空白或崩溃。

---

## 7. 本期不实现功能

- 多用户协作或内容共享
- 文章收藏、标记已读等个人管理功能
- 关键词智能推荐
- 历史推送记录查询
- 多时间段推送（本期仅支持每天一次）
- 公众号内容全文展示（仅展示摘要与跳转链接）
