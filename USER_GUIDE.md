# 公众号内容聚合助手

一款轻量化的微信小程序，支持用户自定义关键词与公众号监控源，定时聚合相关最新文章简报，并通过微信服务通知在指定时间推送给用户。

## 功能特性

### 核心功能
- ✅ 自定义关键词监控（最多10个）
- ✅ 自定义公众号源（最多20个）
- ✅ 每日定时内容聚合
- ✅ 微信服务通知推送
- ✅ 按关键词分组展示文章
- ✅ 手动刷新获取最新内容

### 用户体验
- ✅ 微信授权登录 + 用户名密码登录
- ✅ 工具理性风设计，简约高效
- ✅ 一键初始化预设配置
- ✅ 推送时间自定义设置

## 快速开始

### 1. 注册账号
- 打开小程序，选择"立即注册"
- 输入用户名和密码（用户名仅支持字母、数字和下划线）
- 勾选用户协议和隐私政策
- 点击"注册"按钮

### 2. 初始化配置
- 进入"配置"页面
- 点击"一键初始化"按钮，自动添加预设的关键词和公众号
- 或手动添加自定义关键词和公众号

### 3. 获取内容
- 进入"简报"页面
- 点击"刷新"按钮，系统会自动抓取最新文章
- 按关键词分组查看文章列表
- 点击文章可复制链接

### 4. 设置推送
- 进入"配置"页面，点击"推送设置"
- 设置推送时间（默认18:00）
- 开启推送开关
- 管理通知权限

## 页面说明

### 简报页（首页）
- 展示当日聚合的内容简报列表
- 按关键词分组展示文章
- 每个关键词下最多5条文章
- 支持手动刷新
- 文章信息：标题、公众号名称、发布时间、摘要

### 配置页
- 关键词管理：展示、新增、删除（上限10个）
- 公众号管理：展示、新增、删除（上限20个）
- 一键初始化功能
- 推送设置入口

### 推送设置页
- 展示当前推送时间
- 支持修改推送时间（0-23点）
- 推送开关
- 通知权限管理

### 我的页面
- 展示用户信息
- 推送设置入口
- 退出登录功能

## 使用说明

### 关键词管理
1. 进入"配置"页面
2. 在"关键词管理"区域点击"添加关键词"
3. 从预设关键词中选择（AI、科技、互联网、编程、设计）
4. 点击删除图标可删除关键词

### 公众号管理
1. 进入"配置"页面
2. 在"公众号管理"区域点击"添加公众号"
3. 从预设公众号中选择（量子位、赛博禅心、AI前线、机器之心、新智元）
4. 点击删除图标可删除公众号

### 内容刷新
1. 进入"简报"页面
2. 点击顶部"刷新"按钮
3. 系统会自动抓取最新文章并展示

### 推送设置
1. 进入"配置"页面，点击"推送设置"
2. 设置推送时间（默认18:00）
3. 开启推送开关
4. 点击"管理通知权限"授权微信服务通知

## 技术架构

### 前端技术栈
- Taro 框架（支持微信小程序和H5）
- React + TypeScript
- Tailwind CSS（工具理性风设计）

### 后端技术栈
- Supabase（数据库 + 认证 + Edge Functions）
- PostgreSQL（数据存储）
- Row Level Security（数据安全）

### 数据库表结构
- `profiles`：用户资料表
- `keywords`：关键词表
- `accounts`：公众号表
- `articles`：文章表
- `push_records`：推送记录表

## 注意事项

### 登录方式
- 微信小程序环境：支持微信授权登录 + 用户名密码登录
- H5环境：仅支持用户名密码登录

### 内容抓取
- 由于微信公众号API限制，当前版本使用模拟数据
- 点击刷新按钮会生成模拟文章数据
- 实际生产环境需要接入公众号API或第三方数据源

### 推送功能
- 推送功能需要用户授权微信服务通知权限
- 推送时间修改后，次日起按新时间执行
- 若当日无新内容，则不触发推送

### 数据限制
- 关键词数量上限：10个
- 公众号数量上限：20个
- 每个关键词每个公众号最多展示5条文章

## 常见问题

### Q: 如何添加自定义关键词？
A: 进入"配置"页面，点击"添加关键词"按钮，从预设列表中选择。当前版本暂不支持完全自定义输入。

### Q: 为什么看不到文章？
A: 请先确保已添加关键词和公众号，然后进入"简报"页面点击"刷新"按钮。

### Q: 如何修改推送时间？
A: 进入"配置"页面，点击"推送设置"，选择新的推送时间即可。

### Q: 微信登录失败怎么办？
A: 请确保在微信小程序环境中使用，H5环境请使用用户名密码登录。
