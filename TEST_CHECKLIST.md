# 测试执行检查清单

## 使用说明

本检查清单用于指导测试人员执行完整的测试流程，确保所有关键测试点都被覆盖。

**使用方法**：
1. 按照顺序执行每个测试项
2. 在"执行结果"列标记：✅通过 / ❌失败 / ⏸阻塞
3. 记录失败原因和缺陷编号
4. 完成后统计通过率
